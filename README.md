<a name="english"></a>
## :gb: English

### Overview

A web application that uses Google Gemini AI to convert voice recordings into structured, actionable reminders.

### Features

- **Voice Recording** - Record audio memos
- **AI Transcription** - Gemini-powered conversion
- **Structured Output** - Organized reminders
- **Action Items** - Extractable tasks
- **Easy Interface** - Simple to use

### Quick Start

```bash
git clone https://github.com/fracabu/remember-me.git
cd remember-me
npm install
npm run dev
```
